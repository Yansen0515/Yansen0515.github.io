授权
chmod +x test.sh

使用
./test.sh 1 …

$# 可以统计传递参数的个数
$* 以一个字符串显示所有脚本传递的参数

如果传递的参数包含空格，应使用单引号或者双引号将该参数括起来，以便脚本将这个参数作为整体来接收
eg:
if [ -n “$1” ]; then #([]内部两边要有空格)
echo “包含第一个参数”
else
echo “没有包含第一个参数”
fi

$与$@的区别， 前者会将所有的参数以一个字符串显示； 后者会分别显示
eg:
echo “….$*演示……”
for i in “$“; do
echo $i
doen

echo “….$@演示……”
for i in “$@”; do
echo $i
doen

运行命令： ./test.sh 1 2 3

echo命令

1 直接显示字符串： echo “This is a test”
2 显示转义字符： echo “"This is a test"“
3 显示变量：
read name
echo “$name This is a test”

4 显示换行： (加一行空格)
echo -e “YES, \n” #-e 开启转义

5 不换行：
echo -e “OK！ \c”
echo -n “ “ #输出内容不换行
echo * #匹配当前目录下的所有文件， 应该使用 echo “*”

printf 输出命令

需要自己加入换行符号 \n

printf “%-10s %-8 %-4.2f\n”
%-10, 是指一个宽度为10个字符（-表示左对齐）， %-4.2f 指格式化为小数，其中.2指保留两位小数、

%d 对应位置参数必须是十进制整数， 否则会报错
%s 必须是字符串
%c Char字符
%f Float浮点， 默认为支持6为小数

跳出循环

有的时候，想要强制跳出循环， shell有两个函数： break and continue
break: 会跳出所有的循环
continue: 只会跳出当前循环， 其他循环还继续

while :
do
echo -n “输入 1 到 5 之间的数字:”
read aNum
case $aNum in
1|2|3|4|5) echo “你输入的数字为 $aNum!”
;;
*) echo “你输入的数字不是 1 到 5 之间的! 游戏结束”
break
;;
esac
done
