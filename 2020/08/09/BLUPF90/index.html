使用参数文件的命令
运作renumf90：
echo renum.par | renumf90 | tee renum.log

blupf90家族的其他program 类似。

使用shell命令

做基因型数据的QC

### #!/bin/bash
## 创建文件
cd ../
mkdir pregsf90
### 拷贝参数文件
cd NL12yuan
cp renaireml.par ../pregsf90
cd ../pregsf90

### add more options
echo "OPTION callrate 0.95" >> renaireml.par
echo "OPTION minfreq 0.02" >> renaireml.par
echo "OPTION hwe" >> renaireml.par
echo "OPTION high_correlation" >> renaireml.par
echo "OPTION saveCleanSNPs 0.95" >> renaireml.par

### data file change
sed -i 's:renf90.dat:../NL12yuan/renf90.data:g' renaireml.par
sed -i 's:renadd05.ped:../NL12yuan/renadd05.ped:g' renaireml.par
sed -i 's:mmapa.txt:../mapa.txt:g' renaireml.par
## run
echo renaireml.par | preGSf90 | tee pregs.log

做aireml的运行

### #!/bin/bash
cd ../
mkdir airemlf_90
cd pregsf90
cp renaireml.par ../airemlf_90
cd ../airemlf_90

### delete the old opition in .par file
sed -i '/OPTION/d' renaireml.par

###  add new options
echo "OPTION conv_crit 1d-8" >> renaireml.par
echo "OPTION tol 1d-18" >> renaireml.par
echo "OPTION maxrounds 500" >> renaireml.par
echo "OPTION sol se" >> renaireml.par
### run 
echo renaireml.par | airemlf90 | tee aireml.log
mv solutions solutions_ebv
cp solution_ebv ../

做blupf90的运行

### 创建file
cd ../
mkdir blupf9
### 创建新变量，提取估计得到的方差-协方差
vargen=$(sed '5!d' airemlf90.log)
varres=$(sed '7!d' airemlf90.log)
###
cp renaireml.par ../blupf90
cd blupf90
### 以计算得到的方差协方差 代替 原理假设的
sed -i "s/1.000/$vargen/g" renaireml.par
sed -i "s/0.500/$varres/g" renaireml.par

###  delete原有的option
sed -i '/OPTION/d' renaireml.par

###  增加新的option
echo "OPTION SNP_file ../pregsf90/snp.txt_clean" >> renaireml.par
echo "OPTION no_quality_control" >> renaireml.par
###  run
echo renaireml.par | blupf90 | tee blup.log
### results
mv solutions solution_gebv

awk '{if ($2==7) print $3,$4}' solution_ebv > ebv.tem  # 这个是选出个体的ebv， 7表示的是个体ebv的位置

cp ped_original ped.txt

SNP文件

对基因型的代表数字进行转化，

这里转为0 1 2 形式， 0: AA， 1: Aa, 2: aa
awk ‘{print $2}’ snp_original | tr 3 1 | tr 4 1 | tr 0 9 | tr 2 0 | tr 9 2 > snp_gen.temp

awk ‘{print $1}’ snp_original > snp_id.temp

paste -d” “ snp_id.temp snp_gen.temp > snp1.temp

expand snp1.tem > snp.txt # 将TAB转换为空格

sed -i ‘1 d’ snp.txt # 删除标头

map文件

cp mapa_original mapa1.temp
sed -i ‘1 d’ mapa1.temp
awk ‘$1=(FNR FS $1)’ mapa1.temp | awk ‘{print $1, $3, $4}’ > mapa2.temp # 替换标记的编码
awk ‘{print $1, $2}’ mapa2.temp > mapa_id.temp
awk ‘{printf “%i\n”, $3*100000}’ mapa2.temp > mapa_pos.temp
paste -d’ ‘ mapa_id.temp mapa_pos.temp > mapa.txt
rm *.temp

NR AND FNR

NR：表示awk开始执行程序后所读取的数据行数。

FNR：awk当前读取的记录数，其变量值小于等于NR（比如当读取第二个文件时，FNR是从0开始重新计数，而NR不会）。

NR==FNR：用于在读取两个或两个以上的文件时，判断是不是在读取第一个文件。

BLUPF90的参数

OPTION method VCE 在blupf90test中的意思为： Run airemlf90 for variance component estimation (default running blupf90)
