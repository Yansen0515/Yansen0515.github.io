# 陈焱森个人学术主页 GitHub Pages 双语版

这是一个可直接上传到 GitHub Pages 的静态个人学术主页。新版增加了中英文展示，并新增完整论文、共同作者论文、会议报告和博士论文页面。

## 文件说明

- `index.html`：首页 / Home
- `about.html`：个人简介与教育工作经历 / About
- `research.html`：研究方向 / Research
- `projects.html`：科研项目 / Projects
- `publications.html`：论文与著作 / Publications
- `talks.html`：会议报告 / Talks
- `cv.html`：CV 下载页
- `contact.html`：联系方式 / Contact
- `style.css`：网站样式
- `assets/profile.jpg`：已加入的个人照片
- `assets/profile_original.jpg`：原图压缩备份，便于后续重新裁剪
- `files/cv.pdf`：建议放你的 CV PDF

## GitHub Pages 上传方法

1. 解压本压缩包。
2. 进入你的 `用户名.github.io` 仓库。
3. 删除旧文件，或直接覆盖旧文件。
4. 将解压后的所有文件上传到仓库根目录。
5. 确保 `index.html` 在根目录，而不是在某个文件夹里面。
6. 在 `Settings > Pages` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。
7. 等待 1–5 分钟后访问：`https://你的用户名.github.io`。

## 个人照片

本版本已经将你的照片加入到 `assets/profile.jpg`，首页右侧头像区域会自动显示该照片。

如果以后想更换照片，只需要把新照片命名为 `profile.jpg`，上传并覆盖 `assets/profile.jpg` 即可。

## 添加 CV

把你的简历 PDF 命名为 `cv.pdf`，上传到 `files/` 文件夹。
